# Setup Instructions

## JDK Version Used
- JDK 17 (Recommended for most modern Java projects)
- Ensure that `JAVA_HOME` is set correctly and added to your system PATH.

## Running a "Hello World" Program

### Code Example
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

##### Explanation
- Compile: Run javac HelloWorld.java in the terminal.
- Execute: Run java HelloWorld.
- Output: Hello, World!
